# Frontend Quản lý Thiết bị Phòng học

Ứng dụng ReactJS này là phần frontend cho dự án Quản lý Thiết bị Phòng học. Nó cung cấp giao diện cho các chức năng:

- Đăng nhập với role ADMIN, TECHNICIAN, USER
- Dashboard tổng quan
- Quản lý phòng học
- Quản lý thiết bị
- Cập nhật trạng thái thiết bị
- Ghi nhận lịch sử sửa chữa
- Báo hỏng thiết bị
- Quản lý tài khoản (ADMIN)

## Cài đặt

```bash
cd frontend
npm install
npm run dev
```

## Tài khoản mẫu

- ADMIN / Admin@123
- TECHNICIAN / Tech@123
- USER / User@123

## Công nghệ

- ReactJS
- React Router
- Vite
- HTML / CSS / JavaScript

## Ghi chú

Dữ liệu hiện tại là mock và lưu trong bộ nhớ ứng dụng. Kết nối backend có thể tích hợp dễ dàng sau khi backend sẵn sàng.
